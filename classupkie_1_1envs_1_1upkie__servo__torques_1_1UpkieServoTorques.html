Detailed Description

Command servos by torque control.

Action space

The action space is consists of the following targets for each joint:

  • feedforward_torque: feedforward joint torque \(\tau_{\mathit{ff}}\) in [N m].

This makes the agent command joint torques directly:

\[ \begin{align*} \tau & = \tau_{\mathit{ff}} \end{align*} \]

Observation space

This environment has the same observation space as UpkieServos.
